### 진행 기간
- 2022.11.18~2022.11.25

### 프로젝트 개요
- **서비스명: 집바구니 (아파트 실거래가 조회 서비스)**
- **소개: 공공데이터를 활용하여 아파트 실거래가를 조회하고, 찾고 있는 아파트의 주변 인프라 정보 확인과 수치화된 인프라 점수를 통해 아파트 거래에 도움을 주는 서비스**
